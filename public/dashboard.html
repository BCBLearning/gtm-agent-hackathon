GTM-Agent Hackathon 2026

GTM-Agent is an automated lead detection and enrichment system built for the GTM Hackathon 2026. It scans RSS feeds for business opportunities, enriches company data, and generates actionable leads.

Features:

  • Real-time RSS feed monitoring
  • AI-powered lead qualification
  • Company data enrichment
  • Automated email generation
  • Pipeline value estimation
  • Export capabilities (CSV, JSON)

Mode Selection: Choose between Demo (mock data) and Production (real API) modes.

Note: Production mode requires a valid FullEnrich API key.
